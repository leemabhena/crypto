Enigma Machine Simulator

Encrypt text using enigma machine...

This is a client-side Javascript enigma machine simulator. The use can chooose any three rotors from the five (I - V) and can also choose any reflector between A, B or C. The default rotors are rotors (I - III) and the default reflector is reflector B. The user can also set up the key of the machine with 1 corresponding to A, 2 to B, ..., 26 to Z. The default configuration is LEE. The user can also change the ring settings, the default is, AAA or (111) which doesn't change the rotor turnover notches. The user can also change the plugboard settings, note the format should be as follows: AB CB which means A is swapped with B and C is swapped with D, the ordering of the two does not matter. In case of any error, the default settings are used for encryption. When decryting dont forget to make sure the settings are set as they were during encryption,
