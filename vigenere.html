Vigénere Cipher

Encrypt text using substitution...

This is a client-side Javascript tool to encrypt plaintext using the Vigénere Cipher. The default key is CRYPTO and is the one that is used in all cases where the key is not explicitly set. Can select whether to show only letters and hide other characters or show all characters (Note the results will be different in each case since vigenere cipher adds each character of plaintext to key). Can also generate a random key but you have to state the size of key. Note characters which are not letters are not encoded in any way. Select whether to encode or decode the text below.
